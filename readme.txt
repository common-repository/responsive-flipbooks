Plugin provides easy managing process of turnjs flipbooks in wordpress.

== Description ==

Responsive Flipbooks is a plugin that allow to easy manage HTML5 flipping books from Wordpress admin panel. Flipbooks are supporting all platforms and screen resolutions. You can configure its view as double/single page spread and book/magazine design.

To create new responsive flip book you should Add Media from Content type add/edit page.
Then you chose Create Flipbook option and add images as well as for media gallery and in the second step fill flipbook settings (name, style and display).
At the end you got shortcode with arguments based on settings:
1. Page spread: double or single
1. Design: book or magazine
There is also tinymce icon for inserting flipbooks from the list of existing ones.

You can find existing flipbooks under Responsive Flipbooks menu item. For editing and removing flipbooks click on flipbook thumbnail from the list of created books.

== Installation ==

1. Go to Plugins > Add New.
1. Search for "Responsive flipbooks".
1. Click "Install Now".

OR

1. Download the zip file.
1. Upload the zip file via Plugins > Add New > Upload.

Activate the plugin.
